# components/

Reusable UI components — the design system of AIRP.
Components here are not page-specific; they are imported across multiple pages.

## Sub-folders
| Folder | Contents |
|--------|----------|
| `ui/` | Primitive components — Button, Card, Badge, Spinner, Modal, Input |
| `layout/` | Structural components — Navbar, Sidebar, PageWrapper |
| `charts/` | Recharts wrappers — PriceChart, FundamentalsChart, ConvictionMeter |

## Naming convention
PascalCase for all component files: `AgentProgressCard.tsx`, `DebateThread.tsx`
